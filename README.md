> A CLI notes taking app.

## Install

```sh
npm install
```

## Run

```sh
npm run start
```

## Usage

```sh
## Add a new note

node app.js add --title="Buy" --body="Food"

## List all note

node app.js list

## Read a note

node app.js read --title="Buy"

## Remove a note

node app.js remove --title="Buy"

```
